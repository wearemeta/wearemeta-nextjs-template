# Utils

Utility functions and helpers.

## Guidelines

- **Pure functions**: No side effects
- **Well-typed**: Use TypeScript
- **Testable**: Easy to unit test
- **Documented**: Add JSDoc comments

## Examples

- Date formatting
- String manipulation
- Number formatting
- Validation helpers
- Data transformation

## Usage

```tsx
import { formatCurrency } from '@/lib/utils/format';
```
